# 倒车请注意 — 提示词反推 Skill

> 从参考图片反向推导可用于 AI 图像生成模型的提示词，支持 **Nano Banana** / **Nano Banana 2** / **Nano Banana Pro** / **GPT Image 2** 四大模型，针对每个模型的提示词结构和特性差异生成适配提示词，同时提供中英双语输出。

## 功能特点

- **视觉分析**: 11 维度全面解析参考图视觉元素
- **模型适配**: 根据目标模型自动切换提示词风格
- **增强支持**: 支持在原图基础上增加/修改元素或改变风格
- **双语输出**: 中文 + 英文，直接可用

## 支持的模型

| 模型 | 提示词风格 | 分辨率 | 思考模式 |
|------|------------|--------|----------|
| Nano Banana | 叙述性描述 | 1K | - |
| Nano Banana 2 | 叙述性+分辨率/文字 | 1K-4K | minimal/high |
| Nano Banana Pro | 叙述性+专业参数 | 1K-4K | high |
| GPT Image 2 | 结构化 Brief | 标准 | - |

## 交互流程

上传参考图 → 视觉分析 → 选择目标模型 → 确认增强需求 → 双语提示词输出

## 文件结构

```
倒车请注意/
├── SKILL.md                  # Skill 主文件（交互流程+适配规则）
└── references/
    ├── models-guide.md       # 四大模型特性与提示词规则详细参考
    └── prompt-templates.md   # 提示词模板、完整示例与双语输出格式
```

## 使用方式

1. 安装到 WorkBuddy Skill 目录: `~/.workbuddy/skills/倒车请注意/`
2. 在对话中上传图片并提及反推需求
3. 技能会引导你完成四步交互
